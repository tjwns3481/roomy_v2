# Upsell 설정 페이지 수동 테스트 플로우

## 사전 준비
1. Business 플랜 사용자로 로그인
2. 가이드북 하나 이상 생성

## 테스트 시나리오

### 1. Upsell 설정 페이지 접근
- URL: `/editor/[guidebook-id]/upsell`
- 예상: Business 플랜이면 설정 페이지 표시
- 예상: Free/Pro 플랜이면 업그레이드 안내 표시

### 2. 아이템 추가
1. "아이템 추가" 버튼 클릭
2. 폼 모달 열림 확인
3. 다음 정보 입력:
   - 상품명: "조식 서비스"
   - 설명: "신선한 로컬 재료로 준비한 아침 식사"
   - 가격: 15000
   - 이미지: 업로드 (선택)
   - 활성화: ON
4. "추가" 버튼 클릭
5. 예상: 목록에 새 아이템 표시, 토스트 알림

### 3. 아이템 수정
1. 아이템 카드에서 편집 버튼 클릭
2. 폼 모달에 기존 데이터 로드 확인
3. 가격을 18000으로 변경
4. "수정" 버튼 클릭
5. 예상: 목록에 변경된 가격 표시

### 4. 아이템 활성화/비활성화
1. 아이템 카드의 스위치 토글
2. 예상: 상태 뱃지 변경 (활성 ↔ 비활성)
3. 예상: 토스트 알림

### 5. 아이템 순서 변경
1. "↑ 위로" 또는 "↓ 아래로" 버튼 클릭
2. 예상: 아이템 순서 변경
3. 새로고침 후 순서 유지 확인

### 6. 아이템 삭제
1. 아이템 카드에서 삭제 버튼 클릭
2. 확인 다이얼로그 표시 확인
3. "삭제" 버튼 클릭
4. 예상: 목록에서 아이템 제거

### 7. 요청 관리 탭
1. "요청 관리" 탭 클릭
2. 예상: 요청 목록 테이블 표시
3. 예상: 통계 카드 표시 (대기중, 확인됨, 취소됨)

### 8. 요청 상태 변경
1. 요청 행의 메뉴 버튼 클릭
2. "확인됨으로 변경" 선택
3. 예상: 상태 뱃지 변경, 통계 업데이트
