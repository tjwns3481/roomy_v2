## Phase 목표

### Phase 0: 프로젝트 셋업
- [ ] Next.js 15 프로젝트 실행 확인
- [ ] shadcn/ui 컴포넌트 렌더링 확인
- [ ] Supabase 연결 확인
- [ ] 모든 마이그레이션 적용 확인

### Phase 1: FEAT-0 인증 시스템
- [ ] 3가지 로그인 방식 동작 (Kakao, Google, Magic Link)
- [ ] 보호된 라우트 접근 제어 동작
- [ ] 프로필 조회/수정 동작

### Phase 2: FEAT-1 상품 기능
- [ ] 상품 목록 페이지네이션 동작
- [ ] 상품 상세 페이지 렌더링
- [ ] 장바구니 CRUD 동작

### Phase 3: FEAT-1 결제/다운로드
- [ ] 토스 테스트 결제 성공
- [ ] 결제 후 다운로드 권한 생성
- [ ] Signed URL 다운로드 동작

### Phase 4: FEAT-2 관리자 기능
- [ ] 관리자 접근 제어 동작
- [ ] 상품 등록/수정/삭제 동작
- [ ] 주문 상태 변경 동작

## MVP 성공 기준

- 비개발자가 스캘레톤으로 쇼핑몰 구축 가능
- 라이브 후 스캘레톤 다운로드 50회 이상
