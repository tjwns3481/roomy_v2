# Short Term Rental 템플릿 상세 분석

> 1순위 벤치마킹 대상 - 한국 펜션/풀빌라/에어비앤비 숙소에 직접 적용

---

## 1. 템플릿 개요

### 타겟 사용자
- 에어비앤비 호스트
- 독채 펜션 운영자
- 풀빌라 사업자
- VRBO, Booking.com 숙소 운영자

### 핵심 목적
> 게스트가 체크인부터 체크아웃까지 **반복 질문 없이** 모든 정보를 스스로 찾을 수 있게 함

---

## 2. 템플릿 기본 구조 (Table of Contents)

Touch Stay의 Short Term Rental 템플릿은 다음과 같은 **기본 카테고리**로 구성됨:

### 📋 기본 목차 구조

```
1. Welcome (환영)
   └── Welcome Message
   └── About Your Host
   └── Contact Information
   └── Emergency Contacts

2. Before You Arrive (도착 전)
   └── What to Pack
   └── Directions
   └── Parking
   └── Access & Keys

3. House Guide (숙소 안내)
   └── WiFi Information
   └── Heating & Cooling
   └── Kitchen Appliances
   └── Entertainment (TV, Speakers)
   └── Laundry
   └── Outdoor Areas

4. House Rules (숙소 규칙)
   └── Check-in / Check-out Times
   └── Noise Policy
   └── Smoking Policy
   └── Pet Policy
   └── Party Policy

5. Local Area (주변 정보)
   └── Restaurants
   └── Cafes & Bars
   └── Grocery Stores
   └── Things to Do
   └── Transportation

6. Departure (퇴실)
   └── Check-out Procedure
   └── Trash & Recycling
   └── Lost & Found
```

### 3.2 Before You Arrive (도착 전) 섹션

#### 포함 항목
| 항목 | 설명 | 한국 필수 추가 |
|------|------|---------------|
| What to Pack | 준비물 목록 | 수영복, 바베큐 재료 등 |
| Directions | 오시는 길 | **네이버 지도 링크** 필수 |
| Parking | 주차 안내 | 주차 번호, 자동 등록 방법 |
| Access & Keys | 출입 방법 | **도어락 비밀번호** |

#### 한국 특화 추가 항목

```markdown
📍 오시는 길
- 네이버 지도: [링크]
- 카카오맵: [링크]
- 내비게이션 검색어: "제주 OOO펜션" 또는 "제주시 OO동 123-45"

🚗 주차 안내
- 주차 위치: 건물 앞 1~2번 자리
- 주차 등록: 차량번호 자동 인식 (별도 등록 불필요)

🔐 도어락 비밀번호
- 현관문: 1234*
- 숙소 내 금고: 0000#
```

| 서브카테고리 | 토픽 예시 |
|-------------|----------|
| **WiFi** | 네트워크 이름, 비밀번호 |
| **Heating & Cooling** | 보일러, 에어컨 조작법 |
| **Kitchen** | 커피머신, 오븐, 식기세척기 |

| **Outdoor** | 바베큐 그릴, 수영장, 정원 |

#### 한국 펜션 필수 추가 항목 ⭐

```markdown
## 🌡️ 보일러 사용법 (온돌 난방)
1. 거실 벽면 온도조절기 확인

4. 외출 모드 사용 시 동파 주의!

- 보관 중인 반찬 자유롭게 드세요
- 냄새 나는 음식은 밀폐용기에

1. 전원 버튼 3초 누르기
2. 화력 조절: +/- 버튼
3. 잠금: 🔒 버튼 길게
[사진 또는 영상 첨부]

3. 넷플릭스: 홈 > 넷플릭스 앱
4. 유튜브: 홈 > 유튜브 앱
[리모컨 사진 첨부]

- 온수 나오는 데 약 30초 소요
- 최대 사용 인원: 동시 2명
```

### 3.4 House Rules (숙소 규칙) 섹션

#### Touch Stay 기본 구성

| 규칙 | 일반적 내용 |
|------|------------|
| Check-in/out | 15:00 입실 / 11:00 퇴실 |
| Noise | 22:00 이후 정숙 |
| Smoking | 실내 금연, 지정 구역만 |
| Pets | 사전 문의 필요 |
| Parties | 파티/행사 금지 |
| Max Guests | 예약 인원만 숙박 가능 |

#### 한국 펜션 필수 규칙 ⭐

```markdown
## ⏰ 입퇴실 시간
- 체크인: 오후 3시 (15:00) 이후
- 체크아웃: 오전 11시 (11:00)까지
- 얼리체크인/레이트체크아웃: 사전 문의 (추가 요금 발생)

## 🔇 정숙 시간
- 밤 10시 ~ 아침 8시
- 야외 바베큐는 밤 10시까지
- 불꽃놀이, 폭죽 금지

## 🚭 금연 안내
- 실내 전면 금연 (적발 시 청소비 10만원)
- 흡연 구역: 야외 테라스 재떨이 위치
- 전자담배 포함

## 🐕 반려동물
- 소형견(7kg 미만) 1마리까지 가능
- 사전 신고 필수
- 추가 요금: 2만원/박
- 배변 패드 지참 필수

## 👥 추가 인원
- 예약 인원 외 방문객 금지
- 추가 인원 적발 시 1인당 5만원 추가 청구
- 낮 방문은 사전 협의
```

### 3.5 Local Area (주변 정보) 섹션

#### Touch Stay의 강점: Google Places 연동

- 맛집/명소 검색 → 자동 정보 입력
- 지도에 핀 자동 추가
- 영업시간, 전화번호, 리뷰 점수 표시

#### 한국 버전 필수 대체

| Touch Stay | 한국 서비스 |
|------------|------------|
| Google Places | **네이버 플레이스** 연동 |
| Google Maps 핀 | **네이버 지도** 핀 |
| 리뷰 점수 | 네이버/카카오맵 별점 |

#### 한국 펜션 주변 정보 예시

```markdown
## 🍽️ 맛집 추천

### 현지인 맛집
1. **제주흑돼지 본가** ⭐ 4.8
   - 거리: 차로 5분
   - 추천 메뉴: 오겹살, 항정살
   - 네이버 지도: [링크]

2. **해녀의집 해산물** ⭐ 4.6
   - 거리: 도보 10분
   - 추천 메뉴: 전복죽, 해물뚝배기
   - 영업시간: 08:00-20:00

### ☕ 카페
1. **바다뷰 카페** - 인스타 핫플, 노을 맛집
2. **숲속 베이커리** - 수제빵 추천

### 🛒 마트/편의점
- GS25: 도보 3분
- 이마트24: 차로 2분
- 하나로마트: 차로 10분 (로컬 식재료)

## 🏖️ 주변 관광지
1. **협재해수욕장** - 차로 15분
2. **한라산 영실코스** - 차로 30분
3. **성산일출봉** - 차로 1시간
```

### 3.6 Departure (퇴실) 섹션

#### Touch Stay 기본 항목
- Check-out time reminder
- Key return instructions
- Trash disposal
- Final walkthrough checklist

#### 한국 펜션 퇴실 체크리스트 ⭐

```markdown
## ✅ 퇴실 전 체크리스트

### 필수 확인 사항
- [ ] 에어컨/보일러 전원 OFF
- [ ] 모든 창문 닫기
- [ ] 가스밸브 잠금 확인
- [ ] 수도꼭지 잠금
- [ ] 사용한 식기 설거지
- [ ] 침구류 한 곳에 모아두기 (개지 않아도 됨)

### 🗑️ 분리수거 안내
1. **일반 쓰레기**: 검은 봉투 → 현관 앞
2. **음식물 쓰레기**: 전용 봉투 → 음식물 수거함
3. **재활용**:
   - 페트병: 라벨 제거 후 압축
   - 캔: 세척 후 분리
   - 유리병: 세척 후 분리
   - 종이: 묶어서 배출

### 🔑 퇴실 방법
1. 짐 챙기기
2. 체크리스트 확인
3. 현관문 나와서 **자동 잠금** 확인
4. 카카오톡으로 퇴실 알림 보내주세요!

### 📦 짐 보관
- 퇴실 후 짐 보관 가능 (무료, 당일까지)
- 현관 옆 창고에 보관
- 필요시 사전 연락 주세요
```

## 4. 한국 서비스 차별화 포인트

### Touch Stay에 없는 기능들

| 기능 | 설명 | 구현 우선순위 |
|------|------|-------------|
| **카카오톡 발송** | 알림톡으로 가이드북 링크 전송 | ★★★★★ |
| **네이버 지도 연동** | 주변 정보 자동 가져오기 | ★★★★★ |
| **분리수거 가이드** | 한국 특유 분리배출 설명 | ★★★★★ |
| **온돌 난방 설명** | 보일러 사용법 템플릿 | ★★★★☆ |
| **도어락 비밀번호 표시** | 체크인 시점에만 노출 | ★★★★☆ |
| **퇴실 체크리스트** | 체크박스 인터랙션 | ★★★★☆ |
| **한국어 AI 생성** | 자연스러운 한국어 안내문 | ★★★★★ |

### 개선된 UX 제안

1. **도어락 비밀번호**: 체크인 날짜 이후에만 표시 (보안)
2. **퇴실 알림**: 체크아웃 1시간 전 카톡 알림
3. **분리수거 사진**: 지역별 분리배출 사진 템플릿 제공
4. **음성 안내**: 주요 설명 TTS로 제공 (어르신 손님용)
