# Cleaning Guide & SOP 템플릿 상세 분석

> 3순위 (미래 먹거리) - B2B 확장을 위한 직원/청소 알바용 가이드

---

## 1. 전략적 중요성

### 왜 중요한가?

Touch Stay의 Cleaning Guide / SOP 템플릿은 **게스트용이 아닌 직원용**:
- 청소 알바생 교육
- 신입 직원 온보딩
- 위탁 운영 매뉴얼
- 표준 운영 절차 문서화

### 비즈니스 가치

```
문제점:
"사장님이 매번 바뀌는 청소 알바에게
'이불은 이렇게 개세요'라고 설명하기 입 아프다"

해결책:
사진/동영상으로 청소 기준을 문서화하고
링크 하나로 전달 → 반복 설명 제거
```

### 수익 모델 확장 가능성

| 현재 (게스트용) | 미래 (직원용) |
|----------------|--------------|
| Stay Guide (Guest Ver.) | Stay Guide (Staff Ver.) |
| 무료 또는 기본 제공 | **유료 프리미엄 기능** |
| 1개 가이드 | 가이드 2개 필요 (별도 과금) |

## 3. SOP (Standard Operating Procedure) 템플릿

### 3.1 목적

> 숙박 운영의 **모든 절차를 표준화**하여 누가 하더라도 동일한 품질 유지

### 3.2 기본 구조 (2026년 1월 브라우저 확인)

```
📁 Start here (시작하기)
   └── SOP 개요, 목적

📁 Need to know (필수 사항)
   └── 핵심 규정, 주의사항

📁 Procedure (절차)
   └── 📂 Definition of done (완료 기준)
   └── 📂 Step-by-step instructions (단계별 지침)
   └── 📂 Visual instructions (시각적 가이드)

📁 Essential tools (필수 도구)
   └── 필요한 장비, 소프트웨어

📁 Monitoring (모니터링)
   └── 품질 관리, 점검 방법

📁 Provide feedback (피드백)
   └── 개선 제안, 문의 방법
```

### SOP 템플릿 핵심 특징

| 섹션 | 목적 | 한국 서비스 적용 |
|------|------|-----------------|
| Definition of done | 완료 기준 명확화 | 체크리스트로 구현 |
| Step-by-step | 단계별 상세 지침 | 사진/영상 필수 |
| Visual instructions | 시각적 설명 | 한국어 인포그래픽 |
| Monitoring | 품질 관리 | 사진 증빙 시스템 |

### 3.3 핵심 SOP 예시

## 4. 한국 숙박업 필수 SOP 항목

### 4.1 청소 알바 교육용 ⭐

```markdown
## 🧹 청소 알바 온보딩 가이드

### 첫날 교육 순서
1. 이 가이드북 전체 읽기 (30분)
2. 관리자와 현장 투어 (30분)
3. 첫 청소 동행 실습 (2시간)
4. 단독 청소 + 피드백 (1회)

### 준수 사항
- 도착 시간: 체크아웃 30분 후
- 복장: 깔끔한 평상복 + 앞치마 제공
- 소지품: 개인 휴대폰 외 최소화
- 분실물: 발견 시 즉시 사진 + 관리자 연락

### 급여 및 정산
- 시급: 12,000원
- 정산: 월 2회 (15일, 말일)
- 정산 방법: 계좌이체

### 자주 묻는 질문
Q. 청소 시간이 예상보다 오래 걸리면?
A. 관리자에게 연락. 추가 시간 인정됨.

Q. 손님 물건을 파손했을 때?
A. 즉시 관리자 연락. 고의가 아니면 책임 없음.
```

## 5. Touch Stay vs 우리 서비스 차별화

### 5.1 Touch Stay의 한계

| 항목 | Touch Stay | 개선 필요 |
|------|------------|----------|
| 언어 | 영어 | 한국어 |
| 체크리스트 | 정적 텍스트 | 인터랙티브 체크박스 |
| 완료 보고 | 별도 연락 | **앱 내 완료 버튼** |
| 사진 증빙 | 없음 | **청소 전후 사진 업로드** |
| 알림 | 이메일 | **카카오톡** |
| 급여 연동 | 없음 | **근무 시간 자동 기록** |

### 5.2 한국 서비스 필수 기능

```markdown
## 필수 구현 기능

### 1. 인터랙티브 체크리스트
- [ ] 체크박스 클릭 가능
- [ ] 완료 항목 자동 저장
- [ ] 미완료 항목 알림

### 2. 사진 증빙 시스템
- 청소 전 사진 (필수)
- 청소 후 사진 (필수)
- 문제 발견 시 사진 + 코멘트

### 3. 완료 보고 버튼
- "청소 완료" 버튼 클릭
- 관리자에게 알림 자동 발송
- 근무 시간 자동 기록

### 4. 카카오톡 연동
- 청소 일정 알림
- 완료 보고 알림
- 긴급 연락

### 5. 급여 관리 연동 (미래 기능)
- 근무 시간 자동 집계
- 월간 정산 리포트
- 세금계산서 발행 연동
```

## 7. 핵심 시사점

### Touch Stay에서 배울 점
1. **실시간 업데이트**: SOP 변경 즉시 팀 전체 반영
2. **버전 관리**: 변경 이력 추적
3. **모바일 접근**: 현장에서 스마트폰으로 확인
4. **콘텐츠 공유**: 여러 숙소에서 동일 SOP 재사용

### 한국 시장 차별화 포인트
1. **인터랙티브 체크리스트**: 진짜 체크 가능
2. **사진 증빙 시스템**: 청소 품질 관리
3. **카카오톡 알림**: 한국 표준 메신저
4. **근무 기록 자동화**: 급여 정산 연계
5. **한국어 SOP 템플릿**: 바로 사용 가능
