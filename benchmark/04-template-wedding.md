# Wedding 템플릿 상세 분석

> 2순위 벤치마킹 대상 - 스몰웨딩/돌잔치로 확장 가능성 검증

---

## 1. 템플릿 개요

### 전략적 중요성

Touch Stay에 'Wedding' 템플릿이 존재한다는 것은:
- ✅ **"모바일 청첩장 같은 가이드"** 컨셉이 글로벌하게 검증된 니즈
- ✅ 숙소 안내 → 행사 안내로 **사용 범위 확장** 가능
- ✅ 한국의 **스몰웨딩, 돌잔치, 칠순잔치** 시장 진출 근거

### 타겟 사용자
- 예비 신랑/신부
- 웨딩 플래너
- 스몰웨딩 장소 운영자 (펜션, 레스토랑)
- 이벤트 기획사

### Touch Stay의 정의
> "디지털 중앙 허브" - 결혼식 관련 모든 정보를 한 곳에 모아 게스트, 업체, 가족과 공유

## 4. 한국 웨딩 시장 적용

### 4.1 한국 결혼식 vs 서양 결혼식 차이

| 요소 | 서양 (Touch Stay) | 한국 |
|------|-------------------|------|
| 시간 | 2~4시간 | 30분~1시간 (웨딩홀 기준) |
| 식사 | 피로연 중심 | 뷔페/한정식 |
| 축의금 | 선물/기프트 레지스트리 | **계좌이체** (현금) |
| 드레스코드 | 매우 중요 | 비교적 자유 |
| 초대 범위 | 친밀한 관계 | 넓음 (회사 동료 등) |
| 주차 | 개별 해결 | **주차 안내 필수** |

### 4.2 한국 웨딩 가이드 필수 구조 ⭐

```
1. 💌 청첩 인사
   └── 인사말

2. 📅 일시/장소
   └── 예식 일시
   └── 예식장 정보
   └── 오시는 길
   └── 주차 안내

3. ⏰ 식순 (타임라인)
   └── 예식 시간표
```

## 9. 핵심 시사점 요약

### Touch Stay Wedding에서 배울 점
1. **Memo 기능**: 시점별 자동 메시지 발송 → 카카오 알림톡으로 구현
2. **Deep Link**: 특정 섹션 바로가기 → UX 참고
3. **멀티미디어**: 사진/영상/가상투어 → 필수 구현
4. **통합 플랫폼**: 숙박 + 행사 정보 일원화 → 스몰웨딩 시장 공략

### 한국 시장 필수 차별화
1. **축의금 계좌/카카오페이** → 핵심 기능
2. **카카오톡 공유** → 카톡 최적화 미리보기
3. **한국어 UI/UX** → 완전한 현지화
4. **펜션 연계** → B2B 시장 선점
