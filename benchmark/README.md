## 🎯 벤치마킹 핵심 요약

### Touch Stay란?
영국 기반의 디지털 게스트 가이드북 SaaS 플랫폼. 숙박 시설 정보를 모바일 친화적 웹 앱 형태로 제공하여 게스트 경험을 개선하고 호스트의 반복 문의를 줄여줌.

### 주요 수치
- **사용자 만족도**: 92%가 시간 절약 체감
- **인수**: 2024년 5월 Itrinity에 인수됨
- **가격**: $99/년 (첫 번째 숙소)

## 📊 템플릿 우선순위

### 1순위: 펜션/글램핑 (한국 메인 타겟)
- **Short Term Rental**: 에어비앤비, 독채 펜션, 풀빌라
- **Glamping**: 글램핑장, 감성 캠핑

#### 한국 특화 필수 항목
- 분리수거 방법 (한국 특유)
- 보일러/온돌 난방 사용법
- 인덕션/가스레인지 사용법
- 도어락 비밀번호
- 화로대/불멍 사용법 (글램핑)
- 벌레 퇴치법 (글램핑)

### 2순위: 웨딩 (확장 시장)
- **Wedding**: 스몰웨딩, 펜션 웨딩
- **확장**: 돌잔치, 칠순잔치

#### 한국 특화 필수 항목
- 축의금 계좌번호 + QR코드
- 카카오페이 송금 연동
- 셔틀버스 안내
- 식순/타임라인

### 3순위: B2B (미래 먹거리)
- **Cleaning Guide**: 청소 알바 교육
- **SOP**: 표준 운영 절차

#### 비즈니스 가치
- Guest Guide + Staff Guide 번들 판매
- 추가 수익원 확보

## 🚀 실행 권장사항

### MVP 필수 기능 (Phase 1)
1. ✅ 한국어 가이드북 생성기
2. ✅ 카카오톡 공유/알림
3. ✅ 네이버/카카오 지도 연동
4. ✅ 기본 템플릿 3종 (펜션, 글램핑, 웨딩)
5. ✅ 무료 플랜 제공

### 차별화 기능 (Phase 2)
1. ✅ AI 가이드북 1분 생성 (한국어)
2. ✅ 분리수거 가이드 템플릿
3. ✅ 도어락 비밀번호 시간 제한
4. ✅ 축의금 계좌 QR 기능
5. ✅ 화로대/불멍 가이드

### 성장 기능 (Phase 3)
1. ✅ 야놀자/여기어때 API 연동
2. ✅ Staff Guide (청소/SOP)
3. ✅ 고급 분석 대시보드
4. ✅ B2B 기업용 플랜
